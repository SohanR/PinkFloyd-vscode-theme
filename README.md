## Installation

1.  get the theme from [marketplace](https://marketplace.visualstudio.com/items?itemName=sohanr.pinkfloyd-theme) or
2.  run this command

                    ext install sohanr.pinkfloyd-theme

3.  or go to Extension on VSCode and search `Pink Floyd`.
